The following task:

Test a form with the fields "Day," "Month," and "Year."

You can enter two digits in Day and Month, and four digits in Year. No letters or special characters are allowed.

If all fields contain a value, a message will appear: "correct" or "incorrect."

Folgende Aufgabe:

Eine Maske mit den Feldern „Day“, „Month“ und „Year“ gilt es zu testen.
In Day und Month kann man zwei Ziffern eingeben und in Year vier Ziffern. Keine Buchstaben oder Sonderzeichen.
Wenn alle Felder einen Wert haben, erscheint eine Meldung: „korrekt“ oder „nicht korrekt“.
